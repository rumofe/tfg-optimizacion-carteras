# 📈 Plataforma de Análisis y Optimización de Carteras de Inversión

Bienvenido al repositorio oficial de mi Trabajo de Fin de Grado para el Grado en Ingeniería Informática.

Este proyecto es una solución web integral diseñada para democratizar el análisis financiero avanzado. Ante la sobresaturación de opciones y el "ruido" del mercado actual, esta plataforma transforma la incertidumbre en información estructurada cuantitativa. Está adaptada a la particularidad fiscal española (diferimiento fiscal en Fondos de Inversión), ofreciendo al inversor particular herramientas de soporte a la decisión habitualmente reservadas a profesionales.

## 🚀 Funcionalidades Principales

* **Gestión de Datos Financieros:** Ingesta y normalización de datos procedentes de APIs, incluyendo el procesamiento de informes periódicos para extraer la composición real de los fondos.
* **Motor de Optimización:** Algoritmos de recomendación para generar carteras eficientes basadas en el perfil de riesgo y restricciones configurables (volatilidad máxima, peso por activo).
* **Análisis Visual (X-Ray):** Desagregación total de la cartera para visualizar riesgos solapados y exposiciones geográficas o sectoriales reales.
* **Módulo de Backtesting:** Simulación del comportamiento de la cartera propuesta frente a escenarios históricos de estrés financiero.

## 💻 Stack Tecnológico

* **Backend:** Python (Lógica de negocio, algoritmos de optimización y procesamiento de datos).
* **Frontend:** JavaScript / TypeScript (Desarrollo de estructura web, dashboards e interfaz de usuario).
* **Control de Versiones:** Git y GitHub.
* **Entorno de Desarrollo:** Visual Studio Code.

## 🗓️ Metodología

El desarrollo sigue un enfoque ágil basado en el marco de trabajo SCRUM, con sprints iterativos de 2 a 3 semanas para asegurar una adaptación continua en la integración de datos financieros.
